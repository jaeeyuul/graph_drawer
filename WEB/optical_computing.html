광컴퓨팅(Photonic Computing) 핵심 정리

빛(광자)의 위상·진폭·편광을 정보로 사용해 연산을 수행하는 컴퓨팅 패러다임. 전자의 이동 대신 간섭과 굴절을 이용해 초고속·저지연·저전력 연산을 지향한다.

MZI Mesh Ring Resonator Silicon Photonics Ising Machine Ternary Optical Computer
핵심 한 줄: 광학은 선형 시스템에 강하다 → y = W x 형태의 대규모 행렬 연산을 수동적 간섭망으로 물리적으로 구현 가능.

빠른 키 포인트

  • 광 간섭망(MZI mesh)으로 행렬 곱을 하드웨어에 매핑
  • 링 공진기/위상변조기로 가중치 조절
  • SOA/레이저 기반으로 비선형성 확보(필요 시)
  • 아이징 머신은 최적화 문제를 물리적 완화로 근사
  • 3진수(TOC) 등 다진수 논리 시도 존재

1) 개요

광컴퓨팅은 전자 이동 대신 광자의 전파·간섭을 이용해 정보를 처리한다. 반도체 레이저/변조기/도파로(파장가이드)로 구성된 집적 광학 회로(실리콘 포토닉스 등) 위에서 동작한다. 메모리·비선형 소자가 제한되므로 선형 연산(특히 대규모 행렬)에 자연스럽게 강점을 보인다.

왜 지금 광컴퓨팅인가?
  • 데이터 이동 비용(메모리-연산 간 \u2192 메모리 벽)이 병목
  • 광은 낮은 대역폭 지연·다중화(WDM)·장거리 손실 이점
  • AI 학습/추론의 대규모 y = Wx 패턴이 광학 물리와 잘 맞음

2) 핵심 소자

MZI (Mach–Zehnder Interferometer)

두 경로 위상차로 출력을 조절하는 기본 간섭 소자. 가중치(위상) 조정의 기본 블록.

2x2 빔스플리터 + 위상 시프터 + 2x2 결합기

Ring Resonator

공진 조건에서 특정 파장만 강하게 결합/차단. 필터링·가중치·메모리 유사 저장시간(한정).

Phase Modulator

전압·열·광에 의해 위상을 바꿈. 실리콘 포토닉스에선 PN/PiN, 열광학 방식이 흔함.

SOA (Semiconductor Optical Amplifier)

광 증폭 소자. 포화 효과로 비선형성 제공. SNR/잡음·열설계 고려 필요.

Photonic Crystal

주기적 굴절률로 밴드갭 형성. 초소형·고Q 공진기/웨이브가이드 구현.

Silicon Photonics

CMOS 공정 호환 집적광학 플랫폼. 전자·광 통합이 용이.

3) 연산 패러다임

선형광학 행렬연산

MZI mesh (Clements/Reck 구조)로 임의의 유니터리 행렬을 근사 구현. 가중치=위상.

  • 장점: 대역폭·병렬성(WDM, 공간 다중화)
  • 과제: 재구성 오류, 온도 드리프트, 캘리브레이션

동역학적/아날로그 계산

레이저 네트워크/커플드 공진기 등에서 안정상태를 해로 해석. 예) 광 아이징 머신.

광 논리(불 대수)

SOA 게인 포화/간섭을 이용한 AND/OR/NOT. 대규모 스케일은 손실·누적오차 고려.

다진수 논리 (예: 3진수)

강도·위상·편광 등 다값 신호를 디지털화. 예시: Ternary Optical Computer(TOC) 개념.

4) 장단점 요약

장점설명메모
대역폭/병렬성 WDM·공간 다중화로 대규모 동시 연산 통신/AI 가속에 적합
저지연·전송 손실 우수 포토닉 링크의 전파 속도·장거리 이점 칩간/랙간 인터커넥트 강점
비선형성/메모리 한계 순수 광학으로 강한 비선형 구현이 어렵다 전자 하이브리드로 보완
정밀 제어 필요 열 드리프트/공정 변동성으로 캘리브레이션 비용 피드백·자체보정 회로 필요
범용성 제한 선형 연산 편향. 범용 컴퓨팅 대체는 아직 이르다 특화 가속기로 포지셔닝

5) 응용 분야

AI/신호처리 가속

광학 MAC(곱-누산)으로 CNN/변환기 형태의 대형 행렬 연산 가속.

최적화/조합문제

Ising 모델을 이용해 맥스컷, VRP 등 근사해 탐색.

데이터 센터/인터커넥트

광 스위칭/라우팅, 랙 간 저지연 고대역 폭 링크.

센서·계측

간섭계 기반 고감도 측정, LIDAR 보조 연산.

7) 용어 사전 (핵심만)

  • MZI: 두 경로 간 위상차로 간섭 출력을 제어하는 간섭기
  • Ring Resonator: 고Q 공진기로 파장 선택/지연/가중치 구현
  • SOA: 반도체 광증폭기. 게인 포화·비선형성 제공
  • WDM: 파장 다중화. 여러 파장을 한 도파로로 동시에 전송
  • Silicon Photonics: CMOS 기반 집적 광학 플랫폼
  • Clements/Reck: 유니터리 행렬을 MZI 배열로 분해하는 방식
  • Ising Machine: 이징 모델의 에너지 최소화로 조합최적화 근사
  • TOC: Ternary Optical Computer, 3진 논리 광컴 개념
